# Uber Analytics Dashboard

## Project Overview

Developed an interactive Power BI dashboard to analyze Uber ride operations, revenue trends.

## Features

### Overview Dashboard
- Completed Bookings
- Lost Bookings
- Revenue
- Total Distance
- Average Distance
- Monthly & Quarterly Trends
- Vehicle-wise Revenue
- Top Pickup & Drop Locations
- Driver & Rider Ratings

### Vehicle Analysis
- Booking Count by Vehicle
- Revenue by Vehicle
- Contribution Analysis

### Revenue Analysis
- Revenue by Customer
- Revenue by Vehicle
- Revenue by Payment Method
- Monthly & Quarterly Revenue Trends

## Tools Used
- Power BI
- DAX
- Power Query
- Data Modeling
- Data Visualization
